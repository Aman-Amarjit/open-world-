Core system modules

🚗 Vehicle system

  • Rigidbody with friction, mass, engine torque
  • Steer angle → rotation lerp
  • Enter/exit radius trigger
  • Damage model (4 panels + engine)
  • Speed tiers: sedan, muscle, truck, police

🧍 Human AI system

  • Pedestrian wander (navmesh waypoints)
  • Panic flee on gunshot / explosion
  • Carjack victim → exit car, run
  • Gang member → patrol → attack
  • Police officer → pursue → arrest

🔫 Weapons system

  • Hitscan raycast (pistol, SMG)
  • Projectile physics (RPG, grenade)
  • Ammo pool per weapon type
  • Fire rate timer, reload state
  • Equip from ground pickup or garage

📋 Mission system

  • Mission: array of Objectives
  • Types: Reach, Destroy, Collect, Escape, Timer
  • Trigger zones (invisible rect colliders)
  • Branching success/fail paths
  • Score multiplier per completion

💥 Damage & health

  • Player HP + car HP separate
  • On-foot: bullet, explosion, runover
  • Car: collision speed × mass, bullets
  • Fire spread to nearby vehicles
  • Respawn at hospital / busted at police

🗺 Minimap

  • Canvas 2D overlay, 150×150px
  • Dot: player (white), enemy (red), objective (yellow), police (blue)
  • Clip circle, rotate with player heading
  • Blip pulse on objective

⚡ Power-ups

  • Spawn on timer or at fixed positions
  • Types: Health +50, Ammo ×2, Speed 15s, Armor +50, Wanted Clear
  • Bobbing animation, collect radius

🏆 Score & progression

  • Score: kills, missions, stunts, wanted evade
  • Multiplier: combo chain within 10s
  • Level unlock at score threshold
  • Money: spend in garage

Adaptive music system — situation-driven
🌆

Ambient city — calm exploration

Lo-fi hip-hop, jazz-funk, chill synthwave. Slow tempo (70-90 BPM). Background city noise layered under music.

Trigger: no wanted, not in mission, driving or on foot
🎯

Mission briefing / start

Staccato brass stings, synth arp, tense cinematic intro riff (5-8s). Think GTA1's mission jingle.

Trigger: mission accepted → cut to action track
🔫

Combat / gunfight

High-BPM drum & bass or heavy industrial techno (140-170 BPM). Aggressive synths, distorted bass, driving percussion.

Trigger: player fires weapon OR enemy AI enters attack state
🚨

Police chase — low wanted (1-2 stars)

Funky uptempo groove, wah guitar, chase-movie energy. Think 70s cop show theme (110-130 BPM).

Trigger: wanted ≥ 1 and police in pursuit
🚁

Police chase — high wanted (3-5 stars)

Intense orchestral + electronic hybrid. Staccato strings, alarm synths, racing drums (150+ BPM). Escalates every additional star.

Trigger: wanted ≥ 3, SWAT deployed
✅

Mission success

Triumphant brass fanfare (3-4s). Major key, punchy, celebratory. Score tally SFX layered over.

Trigger: all objectives complete → fade to ambient
💀

Mission fail / death

Descending minor trombone glissando (2s, "wah-wah"). Brief silence, then ambient resumes at hospital.

Trigger: player HP = 0 or time expires
🏁

Car radio — in-vehicle

Multiple fictional stations: Classic Rock, Hip-Hop, Electronic, Latin. Each car has a random starting station. Player cycles with R key. Diegetic — fades when exiting car.

Trigger: player enters vehicle → cross-fade ambient to radio
🏙️

Night / district themes

Each map zone has its own ambient variant: docks = dark jazz, suburbs = soft synthwave, downtown = bustling latin, industrial = mechanical drone. Cross-fade on zone entry.

Trigger: player enters zone boundary
Music system implementation

Audio layers

3 simultaneous audio channels: ambient (looping background), music (situation track), sfx (effects pool). Cross-fade between music states over 1-2s using Web Audio API gain nodes.

State machine

MusicManager has states: CALM → MISSION → COMBAT → CHASE_LOW → CHASE_HIGH → SUCCESS → FAIL. Priority order ensures combat overrides chase, chase overrides calm.

Dynamic intensity

Combat track adds/removes stems (drums, bass, lead) based on enemy count nearby. More enemies = fuller mix. Uses looping segments that can be muted/unmuted seamlessly.

Recommended tech stack
Phaser 3 (renderer + physics) Matter.js (rigid body physics) Web Audio API (adaptive music) Tiled (map editor) TypeScript Webpack / Vite
Vehicle class
class Vehicle extends Entity { vx = 0; vy = 0; angle = 0; // physics state driver = null; // Human | null speed; armor; handling; maxSpeed; update(dt) { if (!this.driver) { this.applyFriction(dt); return; } const input = this.driver.getInput(); this.angle += input.steer * this.handling * dt; const accel = input.throttle * this.speed; this.vx += Math.cos(this.angle) * accel * dt; this.vy += Math.sin(this.angle) * accel * dt; this.clampSpeed(this.maxSpeed); this.applyFriction(dt); this.x += this.vx * dt; this.y += this.vy * dt; } onCollision(other, impulse) { const dmg = impulse * 0.1; this.hp -= dmg; if (other instanceof Human) other.takeDamage(dmg * 2); SFX.play('crash', { volume: impulse / 500 }); } tryEnter(human) { // carjacking if (this.driver) this.ejectDriver(); this.driver = human; human.vehicle = this; EventBus.emit('car:enter', this); } }
Human / police AI state machine
const HumanState = { WANDER: 'wander', FLEE: 'flee', PATROL: 'patrol', CHASE: 'chase', ATTACK: 'attack', ARREST: 'arrest' }; class PoliceOfficer extends Human { update(dt) { const dist = distance(this, game.player); switch (this.state) { case HumanState.PATROL: if (dist < 200 && game.wanted > 0) this.setState(HumanState.CHASE); break; case HumanState.CHASE: this.moveToward(game.player, dt); if (dist < 30) this.setState(HumanState.ARREST); if (dist > 400) this.setState(HumanState.PATROL); break; case HumanState.ARREST: game.bustPlayer(); // lose weapons, pay fine break; } } }

Map technical setup

Tile map

Tiled editor, 16×16 or 32×32 tiles. Layers: ground, roads, buildings (solid), props, colliders. Export as JSON, load with Phaser.Tilemaps.

Zone system

Each map divided into named zones (rectangles). Zone entry/exit triggers ambient music swap, police density change, gang territory behavior.

Traffic & pedestrians

Spawn pool: max 20 cars + 30 peds per level. Despawn when 600px+ from player. Road spline system channels AI vehicles to stay in lanes.

Wanted star system
★★★★★1 star — Patrol cars notice you. 1-2 officers respond.
★★★★★2 stars — Roadblocks appear. Officers on foot pursue.
★★★★★3 stars — SWAT vans. Helicopters scan area. Armed pursuit.
★★★★★4 stars — FBI vehicles. Spike strips on roads. Snipers.

Wanted triggers (what raises stars)

+1 star actions

  • Shooting a weapon in public
  • Crashing into a police car
  • Running over a pedestrian
  • Stealing a car (if witnessed)

+2 star actions

  • Killing a civilian
  • Destroying a police vehicle
  • Assaulting an officer
  • Running a major roadblock

+3-5 star actions

  • Killing a police officer
  • Bombing a building
  • Multiple officer kills
  • Entering restricted zone at 3+

Losing wanted level

  • Hide in alley, no LOS for 10s
  • Paint-n-spray garage visit
  • Collect "police bribe" pickup
  • Evade for 60s continuously
