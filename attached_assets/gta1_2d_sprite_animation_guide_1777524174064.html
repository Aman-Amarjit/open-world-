2D sprite art — GTA 1 style
All drawn with Canvas 2D paths — no shapes, no 3D models, no clip art. Every sprite is pure bezier/lineTo art.

Top-down car anatomy

Each car is drawn as layered Canvas paths: body silhouette (outer hull with subtle asymmetry), windshield (translucent tinted fill), roof panel (darker fill, slightly smaller), wheel arches (4 subtle indentations), wheels (dark ellipses at 4 corners), headlights (small bright ellipses front), taillights (red ellipses rear), side detail lines (1px strokes for door seams).

Rotation system

Draw each car sprite at 0° (facing right). Rotate with ctx.rotate(angle). For pixel-art feel, pre-render 16 or 32 rotation frames into offscreen canvases and pick the nearest frame at runtime — avoids sub-pixel blur. Use Math.round(angle/(Math.PI*2)*16)*frame_step to snap to nearest frame.

Damage states

Draw 3 overlapping damage layers: undamaged (clean body), damaged (dented path variants, cracked windshield lines, dark smoke patch), on fire (flickering orange alpha overlay, animated). Toggle by drawing additional paths on top, no new assets needed.

Human sprite layers

Top-down human: shadow ellipse (semi-transparent, 1px below sprite), legs/feet (two small rounded teardrop paths), torso (tapered oval), arms (thin curved paths, one slightly raised), head (slightly asymmetric ellipse — not a perfect circle), hair (organic blob path on top of head).

Walk animation

8-frame walk cycle. Key frames: neutral stance → left leg fwd/right back → neutral → right leg fwd/left back. Arms swing opposite to legs. Animate leg offsets with sin(t). Head bobs very slightly (+1px). At 12fps feels smooth. Draw each frame into offscreen canvas, cycle by frame index.

Character variants

Same base path, different fill colors and details: Player unique jacket color, gun hand. Gang darker clothes. Police blue uniform, cap. Pedestrian varied shirt/trouser combos. SWAT tactical vest outline, helmet.

Police cruiser markings

Same car body path as sedan, with: white/black two-tone body fill (split with a diagonal path), roof lightbar (narrow rect with alternating red/blue fills, flashes via JS setInterval), "POLICE" text rendered at tiny scale, side door stripe (thin diagonal path), push-bar at front (2 extra line segments).

Siren light animation

Lightbar flashes red then blue alternately at 200ms interval. Draw two small ellipses on roof. Toggle fillStyle between #E24B4A and #378ADD. Add a faint colored halo with a slightly larger semi-transparent ellipse underneath. All Canvas 2D, no CSS effects.

SWAT / helicopter

SWAT van: wider body path, roof hatch detail, black trim. Helicopter (top-down): main rotor = 2 thin elongated ellipses crossing at center, animated rotation. Fuselage = flattened teardrop path. Skids = 2 thin line segments. Tail rotor = small X. Shadow below rotates at same speed.

FPS: 12

Explosion animation

12-frame explosion. Frames 1-3: bright white-yellow core flash. Frames 4-8: expanding orange fireball (bezier circle with jagged outer path for spikes). Frames 9-12: dark smoke clouds (grey organic blobs drifting up, fading alpha). All drawn procedurally — no sprites needed.

Gunfire / muzzle flash

3-frame flash at gun barrel. Frame 1: bright yellow-white star shape (6 triangular spikes path). Frame 2: smaller orange core. Frame 3: gone. Show at exactly gun barrel world position. Bullet: fast-moving 2px wide line segment, drawn and erased each frame.

Skid marks & tire tracks

Persistent Canvas layer. When car speed > threshold and steering angle > 20°, draw dark grey semi-transparent short line segments at each rear wheel position. Line width = tire width (~4px). Accumulate on a static background canvas that never clears — marks persist on road until level reset.

Road system

Asphalt fill: #3A3A3A. Lane markings: dashed #E8C84A center lines (12px dash, 8px gap). Sidewalk: #8B8070 strip 18px wide. Curb: 2px #5A5040 line at edge. Crosswalk: white stripes 6px wide, 4px gap. Draw all as fillRect / fillText-free paths on a background canvas that redraws only when scrolling.

Buildings (top-down)

Building footprint: irregular polygon path (not a perfect rect — slight asymmetry makes it feel hand-drawn). Roof: slightly lighter fill. Roof edge highlight: 1px lighter top/left, 1px darker bottom/right (fake isometric lighting from NW). Windows: tiny dark rect clusters. Doorway: gap in outline with step path.

Shadows & lighting

All objects cast a static offset shadow: semi-transparent dark ellipse drawn before the sprite, offset SE by (4px, 4px). Buildings cast longer rectangular shadow (sun from NW). Night mode: overlay semi-transparent dark blue rect over whole map, then draw circular light cones from streetlamps using radial compositing with globalCompositeOperation.

// Draw a top-down car — no rectangles, all paths function drawCar(ctx, x, y, angle, color, damage=0) { ctx.save(); ctx.translate(x, y); ctx.rotate(angle); // Shadow ctx.fillStyle = 'rgba(0,0,0,0.25)'; ctx.beginPath(); ctx.ellipse(2, 3, 18, 10, 0, 0, Math.PI*2); ctx.fill(); // Body hull — organic path, NOT a rect ctx.fillStyle = color; ctx.strokeStyle = shadeColor(color, -30); ctx.lineWidth = 1.2; ctx.beginPath(); ctx.moveTo(-18, -8); ctx.bezierCurveTo(-20,-8, -20, 8, -17, 9); // rear left ctx.lineTo(10, 10); ctx.bezierCurveTo(18,10, 20,7, 20,4); // front right ctx.lineTo(20, -4); ctx.bezierCurveTo(20,-7, 18,-10, 10,-10); // front left ctx.lineTo(-17, -9); ctx.closePath(); ctx.fill(); ctx.stroke(); // Windshield — tinted translucent ctx.fillStyle = 'rgba(140,200,240,0.55)'; ctx.beginPath(); ctx.moveTo(4, -7); ctx.lineTo(15, -6); ctx.lineTo(15, 6); ctx.lineTo(4, 7); ctx.closePath(); ctx.fill(); // Roof (slightly darker, smaller) ctx.fillStyle = shadeColor(color, -15); ctx.beginPath(); ctx.moveTo(-14,-7); ctx.lineTo(3,-7); ctx.lineTo(3,7); ctx.lineTo(-14,7); ctx.closePath(); ctx.fill(); // Wheels — 4 dark ellipses at corners ctx.fillStyle = '#1a1a1a'; [[-14,-10],[12,-10],[-14,10],[12,10]].forEach(([wx,wy])=>{ ctx.beginPath(); ctx.ellipse(wx, wy, 5, 3, 0, 0, Math.PI*2); ctx.fill(); // wheel rim highlight ctx.fillStyle='rgba(255,255,255,0.25)'; ctx.beginPath(); ctx.ellipse(wx-1,wy-1,2,1.5,0,0,Math.PI*2); ctx.fill(); ctx.fillStyle='#1a1a1a'; }); // Headlights ctx.fillStyle = 'rgba(255,240,180,0.9)'; ctx.beginPath(); ctx.ellipse(20,-5,2,1.5,0,0,Math.PI*2); ctx.fill(); ctx.beginPath(); ctx.ellipse(20,5,2,1.5,0,0,Math.PI*2); ctx.fill(); // Taillights ctx.fillStyle = 'rgba(220,40,40,0.85)'; ctx.beginPath(); ctx.ellipse(-18,-6,2,1.5,0,0,Math.PI*2); ctx.fill(); ctx.beginPath(); ctx.ellipse(-18,6,2,1.5,0,0,Math.PI*2); ctx.fill(); if (damage > 0) drawDamageOverlay(ctx, damage); ctx.restore(); }
// Draw top-down human — legs/torso/head as paths function drawHuman(ctx, x, y, angle, walkFrame, skinColor, clothColor) { ctx.save(); ctx.translate(x, y); ctx.rotate(angle); const t = walkFrame; // 0..7 walk cycle const legSwing = Math.sin(t * Math.PI / 4) * 3; // Shadow ctx.fillStyle = 'rgba(0,0,0,0.2)'; ctx.beginPath(); ctx.ellipse(1,2,5,3,0,0,Math.PI*2); ctx.fill(); // Left leg ctx.fillStyle = clothColor; ctx.beginPath(); ctx.ellipse(-2, 3 + legSwing, 2, 3.5, 0.1, 0, Math.PI*2); ctx.fill(); // Right leg (opposite phase) ctx.beginPath(); ctx.ellipse(2, 3 - legSwing, 2, 3.5, -0.1, 0, Math.PI*2); ctx.fill(); // Torso — tapered organic oval ctx.fillStyle = clothColor; ctx.beginPath(); ctx.moveTo(-4, 0); ctx.bezierCurveTo(-4,-6, 4,-6, 4,0); ctx.bezierCurveTo(3,4, -3,4, -4,0); ctx.fill(); // Arm (right, slightly raised when aiming) ctx.strokeStyle = skinColor; ctx.lineWidth=2; ctx.beginPath(); ctx.moveTo(3,-2); ctx.lineTo(7,-5); ctx.stroke(); // Head — slight ellipse, not perfect circle ctx.fillStyle = skinColor; ctx.beginPath(); ctx.ellipse(0, -8, 3.8, 4.5, 0.05, 0, Math.PI*2); ctx.fill(); ctx.restore(); }
// Animation manager — pre-render frames into offscreen canvases class SpriteAnimator { constructor(drawFn, frameCount, w, h) { this.frames = Array.from({length: frameCount}, (_,i) => { const oc = document.createElement('canvas'); oc.width = w; oc.height = h; drawFn(oc.getContext('2d'), i, frameCount); return oc; }); this.frame = 0; } tick(fps=12) { this.frame = (this.frame + 1) % this.frames.length; } draw(ctx, x, y) { ctx.drawImage(this.frames[this.frame], x, y); } } // 16-direction car rotation pre-render const CAR_ANGLES = 16; const carFrames = Array.from({length: CAR_ANGLES}, (_,i) => { const oc = document.createElement('canvas'); oc.width = 56; oc.height = 56; const ctx = oc.getContext('2d'); ctx.translate(28,28); ctx.rotate(i * Math.PI*2 / CAR_ANGLES); drawCar(ctx, 0, 0, 0, '#C0392B'); return oc; }); // At render time — snap to nearest pre-rendered angle function drawCarRotated(ctx, x, y, worldAngle) { const idx = Math.round(worldAngle/(Math.PI*2)*CAR_ANGLES) % CAR_ANGLES; ctx.drawImage(carFrames[(idx+CAR_ANGLES)%CAR_ANGLES], x-28, y-28); }
